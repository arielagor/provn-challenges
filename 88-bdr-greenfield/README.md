# Challenge 88 — Build a Greenfield Outbound Territory in an Untapped Vertical

**Role archetype:** BDR  
**PROVN page:** https://provn.co/challenge-details/88  
**Submitted:** 2026-04-23

## Brief (paraphrased)

You're the first BDR into a greenfield vertical your company has never sold into. Produce a prospect brief for a named target company with a pitch angle, an account map, a first-touch sequence, and discipline on CRM hygiene and measurement. Explicitly no ZoomInfo / Apollo / Big Four top-accounts as targets — show craft, not a list buy.
