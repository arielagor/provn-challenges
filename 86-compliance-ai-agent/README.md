# Challenge 86 — Automate Compliance Triage With an AI Agent at a Financial Services Firm

**Role archetype:** AI Agent Builder  
**PROVN page:** https://provn.co/challenge-details/86  
**Submitted:** 2026-04-23

## Brief (paraphrased)

Design an AI agent that triages incoming compliance alerts at a mid-size financial services firm. Agent must handle noisy feeds (false-positive-heavy), pull from multiple siloed sources, escalate the real signals to a human compliance officer, and produce an auditable trail of every decision.
